# 1. 프로젝트 개요

## 1.1 개발 배경

교사는 단답형 시험을 만들 때 문제를 직접 선별하고, 시험 후 학생별 답안을 채점하고, 결과를 분석하는 작업을 반복적으로 수행해야 한다. 기존 방식은 문제 관리, 시험지 구성, 답안 채점, 결과 분석이 분리되어 있어 시간이 많이 들고 실수가 발생하기 쉽다.

본 프로젝트는 교사용 단답형 문제은행 기반 데스크톱 앱으로, 문제 등록부터 시험지 생성, 답안 입력, 자동 채점, 결과 분석까지 하나의 흐름으로 처리하기 위해 개발되었다.

## 1.2 프로젝트 목적

본 프로젝트의 목적은 교사가 단답형 시험을 효율적으로 관리할 수 있도록 지원하는 것이다.

주요 목적은 다음과 같다.

- 학생 정보를 등록, 수정, 활성화/비활성화한다.
- 문제은행에 문제를 등록하고 유형, 난이도, 세부 분류별로 관리한다.
- 조건에 따라 문제를 자동 추출하거나 직접 선택하여 시험지를 생성한다.
- 생성된 시험지를 저장하고 PDF로 출력한다.
- 학생별 답안을 입력하거나 CSV로 불러와 자동 채점한다.
- 채점 결과를 바탕으로 문항별, 유형별, 난이도별 분석 결과를 제공한다.
- 대시보드에서 시험 결과 요약과 학생별 결과를 확인한다.

## 1.3 기대 효과

- 시험지 제작 시간 단축
- 학생 답안 채점 자동화로 채점 오류 감소
- 문제 유형별, 난이도별 취약점 파악 가능
- 학생별 성취도 확인 가능
- 문제은행 기반으로 기존 문제 재활용 가능
- 교사의 반복 업무 부담 감소
- 결과 분석 데이터를 통해 수업 보완 방향 도출 가능
