# 7.3 진행 결과

| 영역 | 진행 상태 | 비고 |
| --- | --- | --- |
| 프로젝트 구조 | 진행 중 | MVC 폴더 구조 구성 |
| 학생 관리 | 진행 중 | 등록/조회/수정/비활성화 흐름 검증 필요 |
| 문제은행 관리 | 진행 중 | 문제 CRUD 및 필터링 검증 필요 |
| 시험지 생성 | 진행 중 | 자동 추출, 직접 선택, 미리보기, 저장 흐름 보완 |
| 시험지 삭제 | 진행 중 | 화면 순번이 아닌 exam_id 기준 삭제 필요 |
| PDF 출력 | 진행 중 | 학생용 시험지에서 답안 미노출 확인 필요 |
| 결과 입력 | 진행 중 | 직접 입력 정상, CSV 업로드 상태 반영 검증 필요 |
| 자동 채점 | 진행 중 | exam_id + student_id 기준 저장/조회 유지 필요 |
| 결과 보기 | 진행 중 | 학생별/문항별 상세 데이터 반환 구조 검증 필요 |
| 대시보드 | 진행 중 | 실제 DB 데이터 기반 표시 및 카드 매핑 검증 필요 |
| 분석 화면 | 진행 중 | 전체 정답 시 취약 유형 미표시, 오답률 계산 검증 필요 |
| 문서 | 진행 중 | 요구사항, 테스트 계획, 구현 일정 작성 |
